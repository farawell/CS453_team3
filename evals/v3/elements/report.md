### [rules.txt](./rules.txt)

`````txt
1: The output must contain a section labeled "Company names" followed by a colon and a comma-separated list of company names extracted from the given text. 
2: The output must contain a section labeled "People names" followed by a colon and a comma-separated list of people names extracted from the given text. 
3: The output must contain a section labeled "Specific topics" followed by a colon and a comma-separated list of specific topics relevant to the content of the given text. 
4: The output must contain a section labeled "General themes" followed by a colon and a comma-separated list of general overarching themes relevant to the given text. 
5: Each section in the output should be clearly separated and follow the order: Company names, People names, Specific topics, General themes. 
6: If there are no entities to list in any section, the section must still be present, but it should have an empty list following the colon.
`````


### [inverse_rules.txt](./inverse_rules.txt)

`````txt
7: Company names must not be extracted from the given text, and no section should be labeled for them.  
8: People names must not be extracted from the given text, and no section should be labeled for them.  
9: Specific topics must not be mentioned in any section labeled with such a title.  
10: General themes must not be listed in any section labeled for them.  
11: Each section in the output should be randomly ordered and not clearly separated.  
12: Sections with no entities should be omitted entirely from the output.  
`````
