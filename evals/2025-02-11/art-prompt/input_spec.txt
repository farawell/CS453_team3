The input is a text description that will be used to generate detailed prompts for AI photos.
The input text can contain any combination of characters, including letters, numbers, and punctuation, as long as it forms a coherent description related to the subject, timing, lighting, background, and feeling of the desired photo.
The length of the input text should not exceed 80 words to ensure it can be effectively transformed into a singular paragraph prompt.
The input text must be written in English to guarantee accurate interpretation and generation of AI photos.
