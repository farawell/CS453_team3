### [rules.txt](./rules.txt)

`````txt
1: The output must be a string of HTML formatted text. 
2: The output must contain multiple sentences from the input text, each enclosed in separate <p> tags. 
3: Each <p> tag in the output must contain at least one <strong> tag to emphasize a key word or phrase.
4: Each <p> tag in the output must contain multiple <em> tags to emphasize key words and phrases.
5: The output must include at least three <p> tags to meet the minimum requirement.
6: The content within the <strong> and <em> tags must be relevant key words and phrases from the input text.
`````


### [inverse_rules.txt](./inverse_rules.txt)

`````txt
7: The output must not contain any HTML formatted text.  
8: The output must not contain any sentences from the input text in <p> tags.  
9: Each <p> tag in the output must not contain any <strong> tags.  
10: Each <p> tag in the output must not contain any <em> tags.  
11: The output must not include any <p> tags.  
12: The content within the <strong> and <em> tags must not be relevant key words and phrases from the input text.
`````
